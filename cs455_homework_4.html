    Data preparation
  • Add a new column as class label to the dataset used in homework 3. Use two labels only, with 50 data objects for each label.
    I used A and B in this document as an example.
    Suggestions:
    You can use other non-used attributes to decide the labels. Or use your own judgement to do so.

  • Assign 50 data objects (25 As and 25 Bs) as Training Data, used for constructing the classifier.
    Use the rest 50 data objects as Test Data, used for predicting class labels.

    KNN Classification
  • Implement KNN (K Nearest Neighbor) data classification on your own or use KNN from scikit-learn.
    Demonstrate two K values, i.e., K = 1 and K = 7.

  • Compare the predicted labels against true labels to generate/display the accuracy.

    Decision Tree Classification
  • Train the decision tree using our Training Data (50; 25As and 25Bs)
  • Use the Test Data to test the performance. That is, compare the predicted labels against true labels and compute the accuracy.
